# DS-MAE-YOLOv11-Dermoscopy
Official implementation of the paper: "Robust Non-Invasive Melanoma Early Detection Through Lightweight Self-Supervision". Features DS-MAE pretraining, artifact removal, and two-stage hierarchical classification on HAM10000 dataset.
